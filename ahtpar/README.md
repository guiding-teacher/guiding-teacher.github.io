# منصة التعلم الذكي - المنصة التعليمية المتكاملة

## نظرة عامة

منصة تعليمية متكاملة ومتطورة تجمع بين جميع أدوات التعليم الإلكتروني في مكان واحد. تم تطويرها بأحدث التقنيات لتوفير تجربة تعليمية سلسة وممتعة للمعلمين والطلاب.

---

## المميزات الرئيسية

### 1. نظام المصادقة المتكامل
- تسجيل دخول آمن باستخدام Firebase Authentication
- إنشاء حسابات جديدة للمعلمين والطلاب
- إمكانية المتابعة كزائر
- استعادة كلمة المرور
- المصادقة الثنائية (قريباً)

### 2. أنواع الأسئلة المتعددة
- **اختيار من متعدد**: أسئلة تقليدية مع 4 خيارات
- **صح / خطأ**: أسئلة سريعة للمفاهيم الأساسية
- **ملء الفراغات**: أسئلة تتطلب إجابة نصية قصيرة
- **مطابقة**: ربط العناصر من عمودين
- **أسئلة مفتوحة**: أسئلة مقالية تحتاج تصحيح يدوي

### 3. نظام الاختبارات المتقدم
- مؤقت تنازلي لكل اختبار
- عشوائية الأسئلة والخيارات
- إظهار النتائج الفورية مع التصحيح
- مراجعة الإجابات مع التوضيحات
- لوحة صدارة تفاعلية
- إمكانية تحديد عدد المحاولات
- تحديد درجة النجاح

### 4. الفصول الدراسية الافتراضية
- إنشاء فصول دراسية منظمة
- إدارة الطلاب والمواد
- مشاركة المحتوى داخل الفصل
- تتبع تقدم الطلاب
- تقارير أداء مفصلة

### 5. نظام الواجبات والمهام
- إنشاء واجبات مع مواعيد تسليم
- تقويم تفاعلي لعرض المواعيد
- إشعارات للمواعيد القادمة
- تتبع حالة التسليم
- تصحيح الواجبات

### 6. المكتبة الرقمية
- رفع وتحميل الملفات
- دعم صيغ متعددة (PDF, Word, فيديو, صوت, صور)
- تنظيم الملفات حسب النوع
- معاينة الملفات
- مشاركة الموارد

### 7. نظام التواصل المباشر
- دردشة فورية بين المعلمين والطلاب
- محادثات خاصة وجماعية
- إشعارات فورية للرسائل
- حالة القراءة
- مشاركة الملفات في الدردشة

### 8. نظام الإنجازات والشهادات
- شارات إنجاز متنوعة
- نظام نقاط وترقيات
- 10 مستويات تقدم
- شهادات إتمام قابلة للتحميل
- لوحة صدارة للمنافسة

### 9. لوحة تحكم المعلم
- إحصائيات شاملة
- رسوم بيانية تفاعلية
- تتبع أداء الطلاب
- إدارة المحتوى
- تصدير النتائج

### 10. المميزات التقنية
- تصميم متجاوب بالكامل
- وضع ليلي/نهاري
- رسوم متحركة سلسة
- تحميل سريع
- دعم اللغة العربية
- اتجاه RTL

## التقنيات المستخدمة

### Frontend
- **HTML5**: هيكل الصفحات
- **Tailwind CSS**: التصميم والستايل
- **JavaScript (ES6+)**: المنطق والتفاعل
- **Chart.js**: الرسوم البيانية
- **Font Awesome**: الأيقونات

### Backend
- **Firebase**: Backend-as-a-Service
  - **Firestore**: قاعدة البيانات
  - **Authentication**: المصادقة
  - **Storage**: تخزين الملفات
  - **Hosting**: الاستضافة

### المكتبات الإضافية
- **QRCode.js**: إنشاء رموز QR
- **Canvas Confetti**: تأثيرات الاحتفال
- **SortableJS**: السحب والإفلات

## كيفية الاستخدام

### للمعلمين

1. **إنشاء حساب**
   - اختر "معلم" عند التسجيل
   - أكمل بياناتك الشخصية

2. **إنشاء اختبار**
   - اذهب إلى "إنشاء محتوى"
   - اختر "اختبار تفاعلي"
   - أضف الأسئلة بأنواعها المختلفة
   - حدد الإعدادات (المدة، درجة النجاح...)
   - انشر الاختبار وشارك الرابط

3. **إنشاء فصل دراسي**
   - اذهب إلى "الفصول الدراسية"
   - اضغط "فصل جديد"
   - أدخل اسم الفصل والوصف
   - ادعو الطلاب للانضمام

4. **متابعة الأداء**
   - اذهب إلى "لوحة التحكم"
   - راجع الإحصائيات والرسوم البيانية
   - تابع تقدم كل طالب

### للطلاب

1. **إنشاء حساب**
   - اختر "طالب" عند التسجيل
   - أكمل بياناتك

2. **الانضمام لفصل**
   - اذهب إلى "الفصول الدراسية"
   - ابحث عن فصل معلمك
   - اضغط "انضمام"

3. **حل الاختبارات**
   - اذهب إلى "الواجبات"
   - اختر الاختبار
   - اقرأ التعليمات جيداً
   - اضغط "بدء الاختبار"
   - أجب على جميع الأسئلة
   - راجع إجاباتك ثم سلم

4. **متابعة التقدم**
   - اذهب إلى "الإنجازات"
   - راجع نقاطك وشاراتك
   - تابع مستواك
